## 3. Mô tả chức năng
Hệ thống sử dụng FastAPI để xây dựng một Web API cho phép người dùng gửi ảnh quần áo lên. Hệ thống sẽ sử dụng mô hình AI để nhận diện các loại trang phục có trong ảnh và trả về kết quả dưới dạng JSON.

## 4. Hướng dẫn cài đặt và chạy
1. Cài đặt thư viện: `pip install -r requirements.txt`
2. Chạy chương trình: `uvicorn main:app --reload`
3. Truy cập Swagger UI tại: `http://127.0.0.1:8000/docs`
